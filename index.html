Projects

Parser & Interpreter

The project was divided into 2 parts: Part 1 - writing a parser in Python that takes input in a special language called IPP-code24 and converts it into XML format. Part 2 - writing an interpreter in PHP that takes an XML file generated by the parser as input and executes the commands described in the XML file.

The project was completed as part of the IPP (Principles of Programming Languages) course in the second semester of the third year.

Website for a pub

The goal of the project was to create a website for a pub, without modifying the provided HTML code. All changes had to be made using CSS. The website includes several sections, such as a menu, a list of employees, video interviews with the pub owner, and a table reservation form.

Remote calculator client

The project is a client-server application that allows the user to send mathematical expressions to the server, which evaluates them and sends back the result. The client and server communicate using the TCP/UDP protocol.
